# ezoe
The ezoe command implementation written in Node.js.

## 質問ではない

### Show questions and answers history.

```
$ ezoe
```

### Post question

```
$ ezoe これは質問ですか？
```

### Optsion

You can specify the Ask.fm user's account.

```
$ ezoe -u username
```

## Install

```
$ npm install ezoe -g
```
